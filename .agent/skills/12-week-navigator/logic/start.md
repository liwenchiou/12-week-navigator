
# 🌌 系統啟動工作流 (System Start)

當對話開啟或需要重新對齊狀態時，執行此項：

1. **讀取靈魂文件**：載入並分析 `.agent/instructions.md`。
2. **掛載技能清單**：讀取 `.agent/skills/skills-index.md` 索引。
3. **現狀確認**：
   - 掃描 `workspace/config/` 確認目標配置。
   - 掃描 `workspace/data/` 最新紀錄日期，確認今日位置。

---
**✅ 系統已就緒**

### 🚀 建議開始的步驟：

1. **執行今日紀錄 (habit-tracker)**：
   - **怎麼開始**：輸入「**紀錄今天**」或「**開始紀錄**」。
   - **內容**：啟動每日習慣打卡與能量/情緒紀錄。

2. **執行系統初始化 (init)**：
   - **怎麼開始**：輸入「**執行初始化**」或「**/init**」。
   - **內容**：自主檢查環境依賴、檔案結構完整性（若缺失可歸位樣本）與格式校準。

3. **執行報表總結 (report-guide)**：
   - **怎麼開始**：輸入「**產出報表**」或「**/report-guide**」。
   - **內容**：導引執行週報、月報產出與視覺化網頁生成。

4. **執行補記與修正 (habit-fixer)**：
   - **怎麼開始**：輸入「**補記**」或「**/habit-fixer**」。
   - **內容**：補填缺失日期或修正日誌內容。

5. **執行測試 (skill-tester)**：
   - **怎麼開始**：輸入「**執行測試**」或「**測試 [技能]**」。
   - **內容**：在 `workspace/sandbox/` 目錄下進行功能模擬。

---
**💡 提示**：你可以直接輸入指令關鍵字來啟動對應功能。
