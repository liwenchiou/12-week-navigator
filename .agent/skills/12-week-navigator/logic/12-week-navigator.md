
# 🚀 12 週計畫導覽總部 (Navigator Dashboard)

歡迎使用 12 週計畫管理系統。請根據您當前的需求，選擇以下執行模組：

### 🛠️ 1. 系統初始化 (System Init)
- **指令**：輸入「**執行初始化**」或「**Init**」。
- **用途**：環境診斷、目錄建立、補齊配置範本與跨檔案格式校準。
- **內部對應**：`@/12-week-navigator/workflows/init.md`

### 🧭 2. 啟動對齊導覽 (Start Onboarding)
- **指令**：輸入「**啟動導航**」或「**Start**」。
- **用途**：Agent 狀態喚醒、檢視最新進度、能量評比與今日行動建議。
- **內部對應**：`@/12-week-navigator/workflows/start.md`

### 📊 3. 報表產出導覽 (Report Guide)
- **指令**：輸入「**產出報表**」或「**Report**」。
- **用途**：引導生成週報、月度總結報告與視覺化互動網頁。
- **內部對應**：`@/12-week-navigator/workflows/report-guide.md`

### 📝 4. 補記與紀錄修正 (Habit Fixer)
- **指令**：輸入「**補記**」或「**Fix**」。
- **用途**：修正已存日誌內容、補填過去日期或掃描紀錄空檔。
- **內部對應**：`@/12-week-navigator/workflows/habit-fixer.md`

---
**💡 提示**：此指令為系統唯一旗艦入口，選定後 Agent 將自動讀取套件內部的深度邏輯執行。
