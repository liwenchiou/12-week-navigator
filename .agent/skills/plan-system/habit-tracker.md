# 每日紀錄執行邏輯
當使用者發起「開始紀錄」或「紀錄今日」時，請執行以下互動流程：

1. **讀取配置**：
   - 讀取 `./config/核心習慣.md`。
   - 提取「核心習慣」下方的所有項目。

2. **逐項詢問 (Interative Mode)**：
   - **不要一次列出所有問題。**
   - 請依照清單順序，**一次只問一個習慣**。
   - 例如：「首先，今天喝水有超過 3000ml 嗎？」
   - 等使用者回答後，再問下一個。

3. **靈活判定**：
   - 若使用者中途說「後面都完美」，則剩下的項目直接標註為 ✅。
   - 每一項詢問時，請保持親切、簡短的口吻。

4. **產出日誌與小結**：
   - 所有習慣問完後，請詢問使用者的「本日小結」（生活感性敘述）。
   - 最後將結果存檔至 `/log/daily/YYYYMMDD.md`。
   - 格式與之前規範相同，需包含習慣狀態表與感性敘述。
